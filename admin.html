📋 Step 2 · 方法二：直接貼上圖片網址（最簡單）
不想改 Cloudinary 設定？用這個方法：

1. 登入 Cloudinary → Media Library
2. 點任一張圖片 → 右側面板複製「URL」
3. 重複複製所有圖片的 URL，一行貼一個到下方
4. 點「匯入」，系統自動從 URL 辨識商品編號（A1、B13 等）並忽略中文
🔗 GitHub 自動同步設定
設定後，每次在後台儲存設定都會自動更新 GitHub，前台立即生效！

如何取得 Token（只需做一次）：
1. 登入 GitHub → 右上角頭像 → Settings
2. 左側最底部 → Developer settings
3. Personal access tokens → Tokens (classic)
4. 點 Generate new token (classic)
5. Note 填：sanrio-shop
6. Expiration 選 No expiration（永不過期）
7. 勾選 repo（第一個大項目打勾即可）
8. 點 Generate token → 複製那串 ghp_xxx...

⚠️ Token 只顯示一次，請立即複製貼到下方！
Token 只存在你的瀏覽器，不會上傳到任何地方
